YT 語音轉文字工具

使用 OpenAI Whisper 轉錄 YouTube 影片 → 可選 GPT 整理標點 → 下載 SRT / TXT

① API 設定
🎉 Groq 完全免費，使用 Whisper large-v3，速度快 5-10 倍。 免費申請：console.groq.com → API Keys → Create API Key
② 選擇音訊來源
🖥 NAS 後端（選填） 設定後優先使用自架後端下載，最可靠
— 或直接拖放 / 上傳音訊檔 —

🎵 拖放音訊 / 影片檔案到這裡（可多個）
支援：mp3, mp4, wav, m4a, ogg, webm, flac

③ 轉錄設定
④ 轉錄結果
流程說明： 網頁嘗試透過 cobalt.tools（開源）自動下載 YouTube 音訊； 若失敗請直接上傳音訊檔（可用 yt-dlp 或任何 YouTube 下載工具）。 音訊送至 OpenAI Whisper API 轉錄，再由 GPT 加標點符號整理，最後可下載 SRT / TXT。
API Key 僅儲存在您的瀏覽器，不會傳送給本工具作者。
